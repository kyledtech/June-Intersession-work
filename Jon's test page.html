As a very exciting aside we successfuly were able to deploy our first game without a client release by writing the game logic in Lua using Moonsharp on the client to intereret and execute the scripts. Being able to decouple a game release from a client release is huge in that it allows us to plan our releases a little more flexibly. I can't say that I helped in getting that system going, but I can say that it will make my job a lot easier in the long run. Aside from that I've been working on tools, migriting services from Google Cloud to Microsoft Azure, and getting an alert system running with our ElasticSearch cluster. The last point is a big one for me and I'll have the first version of this system rolled out this week if not the first day of Intercession.
